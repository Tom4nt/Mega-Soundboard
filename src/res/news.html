Version 0.6.0

  • NEW Added an option to allow processing keybinds only when a key is released.
  • NEW Added a seekbar that shows when only one sound is playing.
  • NEW Added a button and keybind to play a random sound in the current Soundboard.
  • NEW Added PTT auto-hold. You can define a set of keys to be virtually pressed while playing sounds.
  • NEW Added a loading screen when launching the app.
  • Paths are no longer checked when creating Sounds, Soundboards or when changing settings.

  • FIX Adding multiple sounds at a time now works properly.

Version 0.5.1

  • FIX Soundboards were still linked to a folder even after being unlinked.
  • FIX First soundboard did not play any sounds until its settings were saved.
  • FIX An error was shown if a key was pressed while closing the app.

Version 0.5.0

  • NEW Tooltips. Tooltips will now appear on most buttons with icons.
  • NEW Folder-linked Soundboards. Soundboards can now be linked to a folder. Sounds are added and removed automatically depending on the contents of the folder.
  • NEW File dragging support. Sounds can now be added by dragging one or multiple sound files directly to a soundboard inside the app window.
  • NEW Move sound option. While adding a sound, the respective file can now be moved to a location specified in Settings.
  • NEW Middle-click on a soundboard to stop all sounds playing from it.
  • FIX Directories are now checked before adding a sound.

Version 0.4.1

  • FIX Scrollbars no longer appear in front of everything.

Version 0.4.0

  • NEW Custom sorting of sounds and soundboards. Drag sounds to reorder them inside a soundboard or drag them to other soundboards to move them! Soundboards can also be reordered.
  • NEW Feedback on playing sounds. Playing sounds now show a little indicator on the top right corder when playing. Click it to stop the sound. Middle-clicking a sound now also stops it.
  • NEW Main info screen. Click on the icon on the top left corner of the app to view info about Mega Soundboard.
  • FIX Mouse lag. Opening another instance no longer causes mouse lag. Mega Soundboard is now limited to one instance.
  • FIX ESC will no longer cause the modal to close while recording a keybind.
